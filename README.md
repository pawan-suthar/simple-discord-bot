
# simple Discord Bot




## Installation

Required modules 

```bash
pip install discord.py
  
```
    
## run code

simply run bot.py in python 

  
## Features


- send hello if someone say hello 
- show ping
- i someone say abusive wods it delete that
- clear messages

## commands
- !ping = show ping
- !clear (no. messages) = delete upper messages
- !pic = send pic
- !link = send a link
- !plzhelp = show all commands
